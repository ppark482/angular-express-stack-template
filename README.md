# Full Stack Template

- Angular JS 
- Bootstrap / Sass / Compass

- Grunt
- Express
- Node
- Nodemon
- bcrypt
- Git / Github 
- Underscore
- Moment
- CLI Color

### Client Side

		$ grunt serve

Defaults to serve up on: 

		localhost:9000

Grunt Server handles:
- SASS Compass
- Compiling SASS into CSS
- Angular and Bower components
- Building out optimized production version
